# Faza 1: Podstawowe komponenty formularzy i nawigacji

## Komponenty formularzy (z #file:c1.md i #file:c2.md)

### Input ✅

- **Plik CSS/SCSS**: `components/_input.scss`
- **Opis**: Style bazowe dla pól tekstowych, warianty, rozmiary, stany, label, ikony, helper text, character counter, clear button, password toggle, accessibility.

### Textarea

- **Plik CSS/SCSS**: `components/_textarea.scss`
- **Opis**: Style bazowe dla textarea, auto-resize, min/max height, resize controls, character counter, wszystkie stany jak w Input, rozmiary wysokości.

### Checkbox

- **Plik CSS/SCSS**: `components/_checkbox.scss`
- **Opis**: Custom checkbox styling, checked/unchecked/indeterminate states, rozmiary, label positioning, disabled/error states, animation, group styling, accessibility.

### Radio

- **Plik CSS/SCSS**: `components/_radio.scss`
- **Opis**: Custom radio button styling, checked/unchecked states, rozmiary, label positioning, disabled state, radio group layout, animation, accessibility.

### Select ✅

- **Plik CSS/SCSS**: `components/_select.scss`
- **Opis**: Custom select styling, arrow icon, multiple select, search/filter, selected items, dropdown positioning, option hover, disabled/error states, group headers.

### Switch (Toggle)

- **Plik CSS/SCSS**: `components/_switch.scss`
- **Opis**: Toggle switch styling, on/off states z animacją, rozmiary, label positioning, disabled/loading states, color variants, smooth transition.

### Form Group

- **Plik CSS/SCSS**: `components/_form-group.scss`
- **Opis**: Kontener dla form fields, label styling, helper/error/success messages, required/optional indicators, field spacing.
